# Claude Code Plugin Marketplace

A collection of plugins for [Claude Code](https://claude.ai/claude-code).

## Installation

To install a plugin from this marketplace, use the Claude Code CLI:

```bash
/plugin marketplace add <plugin-name>
```

Or test a plugin locally by pointing to its directory:

```bash
claude --plugin-dir ./plugins/<plugin-name>
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [tony-soprano-output-style](./plugins/tony-soprano-output-style/) | Makes Claude respond in the voice and mannerisms of Tony Soprano |
